# <a name="stop-a-query-visual-database-tools"></a>Остановка выполнения запроса (визуальные инструменты для баз данных)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
Если очевидно, что выполнение запроса занимает слишком много времени, а ожидаемые результаты не возвращаются, запрос можно остановить, пока он не завершен.  
  
### <a name="to-stop-a-query"></a>Остановка запроса  
  
-   Щелкните правой кнопкой мыши [панель результатов](../../ssms/visual-db-tools/results-pane-visual-database-tools.md), в контекстном меню выберите **Панель**и **Очистить результаты**.
