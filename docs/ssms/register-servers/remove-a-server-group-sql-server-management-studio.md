# <a name="remove-a-server-group-sql-server-management-studio"></a>Удаление группы серверов (среда SQL Server Management Studio)

[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

В этом разделе описывается удаление группы серверов из списка «Зарегистрированные серверы» в [!INCLUDE[ssnoversion](../../includes/ssnoversion-md.md)] с помощью среды [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)]. Всегда можно удалить группу серверов. Если группа серверов не пуста, любые серверы или группы серверов, содержащиеся в удаляемой группе серверов, также будут удалены. Перед удалением группы серверов переместите все серверы и группы серверов, которые нужно сохранить, в новую группу серверов.  
  
##  <a name="SSMSProcedure"></a>  
  
#### <a name="to-remove-a-server-group"></a>Удаление группы серверов  
  
1.  В списке "Зарегистрированные серверы" щелкните правой кнопкой мыши группу серверов, а затем выберите пункт **Удалить**.  
  
2.  В диалоговом окне **Подтверждение удаления** нажмите кнопку **Да**.
