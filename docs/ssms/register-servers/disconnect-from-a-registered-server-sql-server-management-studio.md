# <a name="disconnect-from-a-registered-server-sql-server-management-studio"></a>Отключение от зарегистрированного сервера (среда SQL Server Management Studio)

[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

В этом разделе описывается, как отключить обозреватель объектов от сервера в любое время в [!INCLUDE[ssnoversion](../../includes/ssnoversion-md.md)]. Это не приведет к отключению других компонентов среды [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] , например редактора SQL.  

## <a name="SSMSProcedure"></a>

### <a name="to-disconnect-from-a-registered-server"></a>Отключение от зарегистрированного сервера

- В обозревателе объектов либо щелкните правой кнопкой мыши сервер и нажмите кнопку **Отключиться** , либо нажмите кнопку **Отключиться** на панели инструментов.
