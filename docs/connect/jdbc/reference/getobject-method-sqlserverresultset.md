# <a name="getobject-method-sqlserverresultset"></a>Метод getObject (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Получает значение заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта на языке программирования Java.

|[getObject (int)](../../../connect/jdbc/reference/getobject-method-int-sqlserverresultset.md)|Возвращает значение индекса заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта на языке программирования Java.|  
|[getObject (int, java.util.Map)](../../../connect/jdbc/reference/getobject-method-int-java-util-map-sqlserverresultset.md)|Получает значение заданного индекса столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта языка программирования Java, используя указанный объект Map.|  
|[getObject (java.lang.String)](../../../connect/jdbc/reference/getobject-method-java-lang-string-sqlserverresultset.md)|Возвращает значение имени заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта на языке программирования Java.|
