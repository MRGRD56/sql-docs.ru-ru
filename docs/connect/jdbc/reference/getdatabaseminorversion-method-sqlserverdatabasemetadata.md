# <a name="getdatabaseminorversion-method-sqlserverdatabasemetadata"></a>Метод getDatabaseMinorVersion (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает дополнительный номер версии базы данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public int getDatabaseMinorVersion()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Значение **int**, которое указывает дополнительный номер версии базы данных.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод getDatabaseMinorVersion задается с помощью метода getDatabaseMinorVersion в интерфейсе java.sql.DatabaseMetaData.
