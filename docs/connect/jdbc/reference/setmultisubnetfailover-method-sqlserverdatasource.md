# <a name="setmultisubnetfailover-method-sqlserverdatasource"></a>Метод setMultiSubnetFailover (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Устанавливает значение свойства подключения **multiSubnetFailover**.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
public void setMultiSubnetFailover(boolean multiSubnetFailover);  
```  
  
#### <a name="parameters"></a>Параметры  
 *multiSubnetFailover*  
  
 Новое значение свойства подключения **multiSubnetFailover**.  
  
## <a name="remarks"></a>Remarks  
 Дополнительные сведения о свойстве подключения **multiSubnetFailover** см. в статье о [настройке свойств подключения](../../../connect/jdbc/setting-the-connection-properties.md).
