- [https://docs.microsoft.com/previous-versions/sql/](/previous-versions/sql/)

В них входит документация, по крайней мере, по следующим старым версиям:

- SQL Server 2014
- SQL Server 2012
- SQL Server 2008 R2
- SQL Server 2008
- SQL Server 2005
