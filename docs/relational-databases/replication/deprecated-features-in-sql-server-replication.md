# <a name="deprecated-features-in-sql-server-replication"></a>Устаревшие функции репликации SQL Server
[!INCLUDE [SQL Server SQL MI](../../includes/applies-to-version/sql-asdbmi.md)]
  В этом разделе описаны устаревшие компоненты репликации, по-прежнему доступные в [!INCLUDE[ssnoversion](../../includes/ssnoversion-md.md)]. Эти функции будут удалены в следующем выпуске [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Не следует использовать устаревшие функции в новых приложениях.  
  
## <a name="items-deprecated-in-sssql15"></a>Элементы, нерекомендуемые в [!INCLUDE[ssSQL15](../../includes/sssql16-md.md)]  
  
|Компонент|Описание|  
|-------------|-----------------|  
|[!INCLUDE[ssKatmai](../../includes/sskatmai-md.md)]|Репликация поддерживается, если на каждой конечной точке [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] установлена одна из двух основных версий текущей версии [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Поэтому [!INCLUDE[ssSQL15](../../includes/sssql16-md.md)] не поддерживает репликацию в [!INCLUDE[ssKatmai](../../includes/sskatmai-md.md)] или [!INCLUDE[ssKilimanjaro](../../includes/sskilimanjaro-md.md)]либо оттуда.|  
|[!INCLUDE[ssEW](../../includes/ssew-md.md)]|Репликация поддерживается, если на каждой конечной точке [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] установлена одна из двух основных версий текущей версии [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Поэтому [!INCLUDE[ssSQL15](../../includes/sssql16-md.md)] не поддерживает репликацию в тип данных [!INCLUDE[ssEW](../../includes/ssew-md.md)].|
